🤝 PARTENARIAT COMPTABLE

            
Odoo propose un programme gratuit pour les cabinets comptables et fiduciaires. Une fois inscrits sur la plateforme Odoo, les comptables accèdent aux bases de leurs clients sans coût supplémentaire. L'upsell généré automatiquement se retire de lui-même une fois les 3 conditions remplies — ne jamais le signer.

3 conditions requises :
① Le comptable est inscrit comme fiduciaire agréée sur Odoo② Il est ajouté comme utilisateur dans la base client③ Il se connecte avec le compte Odoo lié à sa fiduciaire
